# Kids School

**⚠️ THIS PROJECT IS UNDER CONSTRUCTION ⚠️**

**Kids School** is a hobby project aimed at developing an interactive learning platform for children. The project focuses on delivering a smooth, engaging experience with modern web technologies.

## Technology Stack

- **Next.js** – Framework for server-side rendering, routing, and optimized performance  
- **TypeScript** – Provides type safety and improves maintainability  
- **GSAP (GreenSock Animation Platform)** – Used for high-performance animations and interactive elements  
- **CSS / Tailwind** – For modular and responsive styling  

## Features

- Animated and interactive learning modules for children  
- Type-safe and scalable codebase for maintainability  
- Optimized for smooth transitions and visual engagement
